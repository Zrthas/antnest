# 蚁巢
## 简介
AntNest（蚁巢）是一款OA工作流WEB软件。借鉴参考市面上常见的OA软件功能，来考验自己的工作中所得知识。

### 主要功能：
   1. **公司内部组织管理【核心】**
      - 成员管理
      - 部门管理
      - 职位管理
   1. **工作管理**
      - 我的工作流
      - 新建工作流
      - 草稿箱
      - 下级工作流
      - 部门工作流
   1. **汇报管理**
      - 日报
      - 周报
      - 月报
   1. **考勤管理**
      - 请假管理
      - 补卡管理
      - 外出管理
      - 出差管理
      - 加班管理
